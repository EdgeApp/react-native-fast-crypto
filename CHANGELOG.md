## 2.1.0 (2022-03-20)

- Remove the OpenSSL-Universal package
- Add a unit-test application
- Upgrade to the Android gradle plugin v3.6.0

## 2.0.0 (2020-11-20)

- Remove the deprecated `default` export. This library uses named ES6 exports.
